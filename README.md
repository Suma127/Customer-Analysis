# 🛍️ Mall Customer Segmentation Analysis

## 📌 Project Overview

This project analyzes mall customer data to identify distinct customer segments based on **Annual Income** and **Spending Score**, along with demographic insights such as **Age and Gender**.

The goal is to understand customer behavior and enable targeted marketing strategies.

---

## 🎯 Objectives

* Analyze customer demographics (Age, Gender)
* Understand income and spending behavior
* Segment customers using clustering techniques
* Provide actionable business insights

---

## 🛠️ Tools & Technologies

* **Python** (pandas, numpy)
* **Matplotlib / Seaborn**
* **Scikit-learn (KMeans)**
* **Jupyter Notebook / Google Colab**

---

## 📂 Dataset Description

The dataset includes:

* `CustomerID` – Unique identifier
* `Gender` – Male / Female
* `Age` – Customer age
* `Annual Income (k$)` – Income in thousands
* `Spending Score (1-100)` – Spending behavior

---

## 🔄 Project Workflow

### 1️⃣ Data Cleaning & Preparation

* Checked missing values and data types
* Standardized column names
* Prepared dataset for analysis

---

### 2️⃣ Exploratory Data Analysis (EDA)

* Age vs Income analysis
* Income vs Spending Score visualization
* Gender-based analysis

---

### 3️⃣ Feature Selection for Clustering

Used:

* Annual Income
* Spending Score

*(Gender was used for analysis, not clustering)*

---

### 4️⃣ Customer Segmentation (KMeans)

* Applied KMeans clustering
* Defined 5 customer segments
* Assigned cluster labels

---

## 📊 Key Insights

* Income and spending behavior vary significantly across customers
* **High-income customers are split into both high and low spenders**
* **Gender-based differences observed in spending patterns**
* The **26–35 age group shows highest earning potential**

---

## 👥 Gender-Based Insights

* Spending behavior varies across genders
* Certain segments show higher concentration of one gender
* Helps in designing **targeted marketing campaigns**

---

## 📈 Customer Segments

| Segment             | Description                    |
| ------------------- | ------------------------------ |
| Premium Customers   | High Income, High Spending     |
| Potential Customers | High Income, Low Spending      |
| Average Customers   | Medium Income, Medium Spending |
| Low Value Customers | Low Income, Low Spending       |
| Impulsive Buyers    | Low Income, High Spending      |
